📍 PCM + JEE + MHT-CET ⏱ Grade 11 → College 🎯 4-Phase Plan
⚡ KEY INSIGHT — JEE IS NOT YOUR ENEMY HERE.
Your JEE prep is secretly building your AI/ML foundation. Every topic you study maps directly to what you'll need later — don't treat them as separate lives.
Calculus (JEE) Gradient descent, backpropagation in ML
Matrices (JEE) Linear algebra = the backbone of all ML
Probability (JEE) Naive Bayes, distributions, stats
Vectors (JEE) Word embeddings, feature spaces
PHASE 01
Foundation Grade 11 · Now → ~Dec 2025
+
🎯 Goal: Python fluency + statistics + data intuition. You already code — this is about becoming dangerous with data, not learning from scratch. Keep it to 5–7 hrs/week alongside JEE.
Python for Data
NumPy Pandas Matplotlib List comprehensions File I/O OOP basics
Math / Stats
Mean / Median / Mode Standard deviation Probability basics Distributions Bayes' theorem
Milestones
Mini Project #1 — "My JEE Data Dashboard" Scrape or manually enter your mock test scores. Use Pandas + Matplotlib to visualize weak areas by chapter. Real data, real problem, 0 cost.
Mini Project #2 — EDA on any dataset Pick any Kaggle dataset (Titanic, Iris). Do full Exploratory Data Analysis — clean, describe, visualize, extract insights. No ML yet.
Resources
📺
Corey Schafer — Python OOP youtube · free · 4 hrs total · best OOP tutorial on the internet
📘
Kaggle's free micro-courses kaggle.com/learn · Python, Pandas, Data Viz · ~15 hrs total · certificate
📺
StatQuest with Josh Starmer youtube · stats + ML concepts explained visually · binge on weekends
PHASE 02
Core ML + Analytics Grade 12 · ~Jan 2026 → ~May 2027
+
⚠️ Reality check: Grade 12 is brutal with boards + JEE/CET. Drop to 3–4 hrs/week during heavy exam months (Jan–April). Protect your exams — they're the actual gate to college. Resume full pace after CET.
Data Analytics Track
SQL Seaborn Plotly Feature Engineering Data Cleaning Storytelling with data Google Sheets / Excel
ML Foundations
Scikit-learn Linear Regression Logistic Regression Decision Trees Random Forest Train/Test split Cross-validation Overfitting / Underfitting
Milestones
Kaggle Competition — "Titanic" or "House Prices" Submit an actual Kaggle entry. Score on the leaderboard. This forces you to think about feature engineering, model selection, evaluation metrics — all in one go.
SQL Project — Local Business Analytics Connect to your Neon PostgreSQL (you already have it). Build a mock "sales analytics" schema. Run aggregations, JOINs, window functions. SQL is the #1 hire-able data skill.
Portfolio Project — Prediction App Build a small Flask/Streamlit app that takes user input and runs a Scikit-learn model. Deploy it free on Railway. This is your first "real" AI project.
Resources
📘
Kaggle ML Course (free) kaggle.com/learn/intro-to-machine-learning · hands-on, no fluff, ~8 hrs
📺
SQLZoo / Mode Analytics SQL Tutorial interactive SQL in browser · free · complete in ~10 hrs total
📘
"Hands-On ML" — Aurélien Géron book · THE bible of practical ML · read chapters 1–8 this phase
PHASE 03
Deep Learning + Specialization Year 1 of College · 2027–2028
+
🔱 Fork point: By now, you choose whether you want to go deeper into Data Analytics / BI (the "insight" path) or into ML Engineering / Deep Learning (the "build models" path). Both are valid. Most people do both at a surface level — you'll go deep on one.
Deep Learning Core
PyTorch Neural Networks CNNs RNNs / LSTMs Attention mechanism Transformers Transfer Learning Fine-tuning LLMs
MLOps + Deployment
Docker basics FastAPI Model monitoring MLflow Hugging Face Cloud (GCP/AWS basics)
Milestones
Karpathy Path — Build Neural Net from Scratch You've already studied micrograd. Revisit it. Then do his "makemore" and nanoGPT series. Understanding the internals separates engineers from users.
Kaggle Expert badge Compete in 5 competitions, earn medals. Expert badge on Kaggle is a legitimate, verifiable credential that hiring managers actually check.
Capstone — End-to-End ML System Full pipeline: data ingestion → preprocessing → training → serving via API → monitoring. Deploy it. This is your hiring portfolio centerpiece.
Resources
🧠
fast.ai — Practical Deep Learning for Coders fast.ai · free · top-down approach · best intro to DL that exists
📺
Andrej Karpathy's YouTube series youtube · "Neural Networks: Zero to Hero" · must-watch, non-negotiable
🤗
Hugging Face NLP Course huggingface.co/learn · free · transformers, fine-tuning, pipelines
PHASE 04
Advanced Research + Industry Year 2–3 of College · 2028+
+
🚀 Endgame: By this phase, you're not following roadmaps — you're reading papers and building original things. The goal shifts from "learn X" to "contribute to X." Open source, research papers, internships, startups — this is where it gets real.
Advanced ML Areas
Reinforcement Learning Diffusion Models Graph Neural Networks LLM fine-tuning (LoRA) RAG systems AI Agents
Career Actions
Open source PRs Research paper reading arXiv daily habit Internships Technical blog Network on LinkedIn/X
Milestones
First Open Source Contribution Pick a library you use (Scikit-learn, Hugging Face, LangChain). Fix a bug, improve docs, add a test. Your GitHub becomes your actual resume.
Replicate a Paper Pick any ICLR / NeurIPS paper and reproduce its results. Write a blog post about what you learned. This signals serious intent to any company or grad school.

The real rule: One project beats ten tutorials. Ship something real every phase — even broken, even small.
The trap to avoid: Tutorial hell. After 20% of a course, build something. Don't wait for permission to start.
Your unfair advantage: You're already deploying real software at 16. Most people starting AI/ML don't know how to ship. You do.
